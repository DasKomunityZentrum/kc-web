KC WEB PROJECT
=================


Requirements
------------

- Web Project for Nette 3.1 requires PHP 7.2
- Composer
- NPM


Installation
------------

- Make directories `temp/` and `log/` writable.
- Run `composer install` in `root` dir of project
- Run `npm install` in `www` dir of project
- Import `database/main.sql` into your database
- Create `config/local.neon` by its template `config/local.neon.template`
- I think its all. Enjoy!
